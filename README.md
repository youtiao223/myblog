## 后端技术栈
- 配置文件yaml解析
  - spf13/viper
- 日志工具
  - logrus
  - file-rotatelogs
  - lfshook
- 用户密码存储
  - **md5加盐加密** (目前使用的)
  - bcrypt
  - scrypt（专家方案）
- 用户身份认证
  - jwt-go
- 表单数据验证
  - go-playground/validator (gin的默认验证器)
- 文件上传
  - qiniu 
- 跨域配置
  - gin-contrib/cors
